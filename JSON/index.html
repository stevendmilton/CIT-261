JSON Objects

JSON Definition

JavaScript Object Notation is an open-standard file format or data interchange format that uses human-readable text to transmit data objects consisting of attributeâ€“value pairs and array data types. It is a very common data format, with a diverse range of applications, such as serving as replacement for XML in AJAX systems. This is important because only text can be transferred between a web browser and a web server. These can be received from the web server and converted into Javascript objects to be manipulated further.

You have two methods you can use when working with JSON data. There is parse which will convert the JSON string data to JavaScript objects. Secondly, you have stringify works in the opposite manner. Stringify will take a JavaScript object and convert it to a text string which can then be sent to the web server to be processed. You are not able to pass dates or functions in JSON. Dates should be passed as text then converted in JavaScript to be used as true dates.

Example of a JSON object

var person = {name:"Steven", age:56, employed:false, interests: ["golf","reading", "tennis"]}

JSON objects are surrounded by curly braces {}.
JSON objects are written in key/value pairs.
Keys must be strings, and values must be a valid JSON data type (string, number, object, array, boolean or null).
Keys and values are separated by a colon.
Each key/value pair is separated by a comma.

To convert the JavaScript object to JSON string, I would do the following:

var personString = JSON.stringify(person);

To convert the JSON text back to a JSON object, I would do the following:

var personObject = JSON.parse(personString);
