JavaScript Objects

Object Definition

An object is a collection of related data and/or functionality which usually consist of several variables and functions which are called properties and methods when they are inside the object. Below is an example of a simple JavaScript object.

const car = {}; This is all that is needed to create an object. This is an empty object so not much use. Let's make it more interesting.

const car = {
make: 'Dodge',
model: 'Charger',
year: '2020',
options: ['4wd','spoiler','supercharger'],
desc: function() {
alert(this.year + ' ' + this.make + ' ' + this.model + ' with options ' + this.options[0] + ', ' + this.options[1] + ', and ' + this.options[2]);
}
}
Now this car has properties and a method to display the car. Next, we will create a constructor function that allows us to create many instances of an Car object.

function Car(make, model,year) { this.make = make; this.model = model; this.year = year; this.displayType = function() { alert(this.year + " - " + this.make + " - " + this.model); }; }

To instantiate the object, use the following syntax:

var honda = new Car('Honda','Accord','2019');

You have two different methods of accessing the class properties. You can either use dot notation or you can use bracket notation.

Dot Notation:
alert(car.make); will return the make of the car

Bracket Notation:
alert(car["make"]); will also return the make of the car

To call the object function, use the following syntax:
car1.desc();
To call the displayType method, you will just use the object name with the dot and the method, like this:

honda.displayType();
